## 📋 完成清单

### 1. ✅ Suno 平台支持
- **manifest.json**: 添加 `suno.com` 到权限列表
- **content.js**: 添加 Suno 平台检测和输入框选择器
- **支持功能**:
  - 歌词输入框
  - 歌曲风格描述输入框
  - 自动检测并显示浮动按钮

### 2. ✅ 音频模式升级 - 歌词/风格选择

#### HTML 更新
在音频面板中添加了类型选择器：

```
┌─────────────────────────────┐
│ 优化目标：                  │
│ ┌──────────┬──────────┐   │
│ │ 🎨 歌曲风格 │ 📝 歌词  │   │
│ │ Suno描述  │ 完整结构 │   │
│ └──────────┴──────────┘   │
└─────────────────────────────┘
```

#### 功能特性
- **默认选择**: 🎨 歌曲风格（适合 Suno 的 Style 输入框）
- **可选**: 📝 歌词创作（生成完整的歌曲结构）
- **智能提示**: 输入框提示会根据选择自动更新
- **美观样式**: 单选按钮卡片，选中高亮

### 3. ✅ 智能提示词生成

#### 🎨 歌曲风格模式
生成适合 Suno 的英文 prompt：
```
A [genre] song with [mood] atmosphere,
featuring [instruments].
Tempo: [BPM], in [style] style.

示例：
A relaxing ambient song with calm atmosphere,
featuring soft piano and gentle percussion.
Tempo: 70 BPM, in lo-fi style.
```

#### 📝 歌词创作模式
生成完整的歌曲结构：

```
[Chorus]

奔跑不会停顿...

...

...
```

## 🎯 使用流程

### 在 Suno 上使用

1. **打开 Suno** (suno.com)
2. **打开 Gestalt** (点击扩展图标或侧边栏)
3. **切换到 🎵 音频模式**
4. **选择优化目标**：
   - 🎨 歌曲风格 → 优化后复制到 Suno 的 "Song Description"
   - 📝 歌词创作 → 优化后复制到 Suno 的 "Lyrics"
5. **输入描述** → 发送 → 复制结果

---

## 📦 更新的文件

### 主要文件
1. **manifest.json**
   - host_permissions: 添加 suno.com
   - content_scripts matches: 添加 suno.com

2. **content.js**
   - 添加 Suno 平台配置
   - 支持歌词和风格描述输入框

3. **sidepanel.html**
   - 添加音频类型选择器
   - 更新音频欢迎消息

4. **sidepanel.css**
   - 添加音频类型选择器样式

5. **sidepanel.js**
   - 添加 audioType 状态变量
   - 添加类型切换监听
   - 添加 getAudioSystemPrompt() 函数
   - 根据类型生成不同提示词
